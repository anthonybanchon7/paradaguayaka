Parada Guayaka - Flask app (generated)
--------------------------------------
Estructura:
- app.py
- templates/index.html
- static/style.css
- static/img/*.png

Instrucciones:
1) Crear y activar un entorno virtual (recomendado):
   python -m venv venv
   source venv/bin/activate    # Linux/macOS
   venv\Scripts\activate     # Windows

2) Instalar dependencias:
   pip install flask pillow

3) Ejecutar la app:
   python app.py

4) Abrir en la tablet (si la corriste en tu PC):
   - Averigua la IP local de tu PC (ej: 192.168.1.10)
   - En la tablet, abre: http://<PC_IP>:5000

Nota: Las imágenes incluidas son marcadores de posición. Reemplázalas en static/img/ con tus PNGs.
